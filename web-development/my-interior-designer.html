MY INTERIOR DESIGNER

Fully Hand Coded

Description

This was one of the last projects that my program had to do before our practicums. The project mainly focuses on back-end development and the functionalities rather than the overall design of the website. Thus, simple flat designs were used throughout the different pages. The core purpose of this project was to create better-designed database tables and use a different type of method saving information on the web. The different methods were local storage, session storage, database, and session variable. Additionally, we have picked interior designer as a topic of our choice due to the fact that it would be enjoyable to create a portfolio website for our fictional designers.

Process

As an end result does not need high quality of design works, my group decided to focus more on the functionalities of the website. However, my group still followed the proper process of creating wireframe and a simple style tile to keep the design consistent and user-friendly. After creating the design, we then created mock up using PHPTAL with XHTML to easily replace the content when the back end information was finished. Additionally using PHPTAL, we were able to show the process of our website to our instructor without setting up the databases nor upload the website online. Moving on to the administration page, we have created adding, removing, and editing product functionalities for the registered admins.

Summary

Overall, this project was a great experience to learn PHP login securities and making your tables as efficient as possible. Furthermore, the first time experience with XML and XHTML libraries was interesting and convenient to use as well. I learned that I should do it more often as it helps to show the work you have done to your clients easier without and databases nor having it uploaded to the website.

Project Details

Category: Web Development

Client: BCIT School Project

Date: 2017 March

Project Type: Team Project

Purpose:

  • Build a web app that allows an admin to manage products with their product web application via a dashboard.

Outcome:

  • Encrypt passwords using Bcrypt.
  • Create a simple shopping cart using session.
  • Make PHP tables to 2nd normalization form.
  • Implement object-oriented programming for the PHP codes for efficiency.
  • Use XML / XHTML template library called PHPTAL.
