NEW HOPE COMMUNITY CHURCH

Wordpress CMS

Description

This was a volunteer project to revamp a church website from an HTML & CSS website to a Wordpress site. At this period I was not experienced with WordPress. Due to time constraint, we had to use a theme and modify it to the clients need.

Process

First of all, we created three different wireframes and mockups that were requested by the client. After the client picked one of the mock up, we then went to find a similar theme on the WordPress website. When we found a matching theme that needed a little adjustment to fit the criteria that the client provided, we then proceeded to customize the theme with CSS and PHP.

Summary

From this volunteer project, I have learned to customize the CSS and PHP code of the website to satisfy our need. Additionally, we have learned to use different plugins to make the website contain unique functionalities compared to the previous version.

Project Details

Category: Web Development

Client: New Hope Community Church

Date: 2016 August

Project Type: Team Project

Purpose:

  • Revamp the old NHCC website using Wordpress themes and custom CSS.
  • Implement different plugins to improve the efficiency of sites functionalities.

Outcome:

  • Modify pre-existing themes to match the clients need.
  • Use different plugins that can create better functionalities. In this website, plugins like image gallery, carousel, and sermons list.
  • Plan and discuss with a real client about different aspects of this project.
