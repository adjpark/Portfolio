TEAM

Fully Hand Coded

Description

What is TEAM? TEAM is a web application that can connect different people together to form a team. The process is very simple; when you register for the application, you can either upload a post to create a team or join a team by applying for existing posts. If you are creating a team, you can simply fill in all the text inputs and create different roles that are required for your team. After the team is created, different users can read the post and can decide to apply to your team. If there are applicants for your team, you can view them on my team page. Furthermore, you can either decide to accept or decline the applicants. When you gather all the members of your team, your post will be deleted automatically.

My group came up with this idea to easily find teammates in the area that you have passions. In this team, I was mainly responsible for the development side which was to code with front and back end languages.

Process

My job as a developer was to meet the deadlines that are set by the project manager and work together with the designers to create a website based on the mockups and wireframes. This project was my first project where different members specialize in different areas. For example, we had members for marketing, project management, and design. By working in a team environment like this, I was able to experience what it feels like to work in a team in a workplace. This project was the first project that required a back-end development. Compared to a front-end development where I excel at, it was very challenging to work in both areas. However, as I got into PHP and MySQL, it was very interesting and enjoyable to learn. During the development phases, some of the extra features that I included were having a profile picture for the team and individual users and a JavaScript oriented search system that compares a string to find a matching result.

Summary

Overall, the hardest part of this project was creating an efficient database and writing clean PHP code. Doing this project gave me more exposure to different development languages. This project strengthened my fundamentals of web coding.
