MONSOX

Fully Hand Coded

Description

Monsox was a product that Aimee Chung our design instructor created during the summer. She wanted to create a working website that can give a brand identity to Monsox. For this project, I was assigned as one of the web developers in the web development team. Other than contributing and helping on the different parts of the website, my main responsibility was creating the contact page for the website.

Process

First of all, all of the members of the web development team were required to create a wireframe for the website. Afterward, we discussed with the design team and picked the best wireframe. Furthermore, we discussed with the design team about what kind of assets would be required for the website. As my main responsibility was to create a contact us page, I created a mock-up version of the wireframe. After the feedbacks and multiple revisions, the final design of the contact us page was created. Next, I have created a PHP function that sends user email to the Monsox email for the contact us page functionality. This was a simple process of using AJAX call to the PHP and running the PHP mail function.

Summary

It was interesting to work with the different teams, and I learned that communication is the key to great teamwork. During the project, some requests were not finished on time due to bad communications between the different teams.

Project Details

Category: Web Development

Client: BCIT School Project

Date: 2016 April

Project Type: Team Project

Purpose:

  • To create a website that has a brand identity.
  • Work in a web development team and perform the tasks assigned by the team lead.

Outcome:

  • Work in an envrionment that consists of different teams such as marketing, development, and design.
  • Create a contact us page that has a custom validation.
  • Use Google map API to customize the map to fit with the website design.
  • Use PHP function that receives an email from the user who completed a contact us form.
