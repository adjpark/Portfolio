MUSIQ

Fully Hand Coded

Description

What is MusiQ? MusiQ was one of my projects from BCIT, where the goal of the project was to create a simple questionnaire which generates a dynamic result to users. The goal of the website was to promote one of my favorite Youtube Channel NoCopyrightSounds. All of the videos are linked to the NCS Youtube channel, and they have been credited appropriately.

There are two main functions for this website. First, when you complete the simple questionnaire, the website will generate an embedded music video that matches to your answers from the questionnaire. Another key aspect of this website is an infographic that was created with SVG, CSS, and JavaScript.

Process

The first step of this website was to create a wireframe and style guide for the website. The wireframe was created with using Balsamiq mockup application. The website is simple and easy to use, and flat design was used for the overall look of the website. For the color scheme, vibrant colors were picked to EDM genre. The name for this website came from adding music and queue together which shortens into MusiQ. The message that I tried to convey with the name is that there is a song that is in the queue for you.

After the wireframe and style guide, questions for the questionnaire were created. For the questionnaire, I created a ranking system to give varying result for different user inputs. Each question can either score a low or high number and depending on the total number it generates different results. Lastly, to create an infographic for EDM genre, a detailed research was done.

After completing the design aspect of the website, it was time to start coding. Media queries and Bootstrap was used for creating a responsive and consistent website. Furthermore, JavaScript was used for all of the functionalities of the website. After completing all the tasks, MusiQ was born.

Summary

One of the difficulties that I faced during this project was exporting Illustrator files into SVG so that I can use it on my website. I learned that proper layer naming convention should be used for the illustrator because all the grouping and naming carries over when you export the file as SVG. Overall, I enjoyed creating a website about a topic that I love.
