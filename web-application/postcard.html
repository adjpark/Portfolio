Postcard

React JS

Description

This project was to get acquainted with React JS. The overall goal was to create a virtual postcard that the user can build by entering the image URL links and text inputs. This virtual postcard will be then saved on the the local storage for users to look at it again in the future. Furthermore, individual users can create multiple postcards, and they can cycle through them.

Process

All the content was created with React, and only HTML content was the single div to print all the classes. A total of three classes was created with were the preview, control, and app. The app class was to render both the control and the preview class and include all the functions that were associated with the other classes. The Preview class was the class that was used to show the actual postcard when all the inputs were submitted using the control class.

Summary

This was the first project for working with React. I found it very interesting to update content without the need to refresh the page. One of the difficulties that I had with this project was to figure out how to set state properly so that I can use the variables in different classes. Overall, this was a fun experience of using React.

Project Details

Category: Web Application

Client: BCIT School Project

Date: 2016 December

Project Type: Individual Project

Purpose:

  • Use React to create a custom user-generated postcard.
  • Save the postcard in the local storage for the user to access it even after closing the browser.

Outcome:

  • Use React js to make the content update automatically without having to refresh the page.
  • Creating different classes in React and putting them together.
  • Implementing functions within different classes and associating with other classes with using state.
  • Use HTML local storage to store all the content for future use.
