Newsfeed

React JS

Description

This project was to create a simple version of newsfeed system that exists on the web. The user would enter their username, image link, country, and message. Then their message will be left on the newsfeed page where other people can see. All of the user input will be automatically updated with React and users can pick their location with google map or login with their facebook info to skip writing the inputs manually.

Process

The content were all rendered through React JS. For this application, a total of two classes was created which are user module and the post module. The user module includes all the inputs, facebook login, and google map. Post module contains the message input and the list of the message that has been posted by other users. Lastly, after all the functionalities have been implemented, all of the messages were saved on to the database using AJAX call so that the messages stay online.

Summary

From this project, I was able to practice using React and implement Facebook and Google API for more functionalities. It was a new experience incorporating APIs into the React JS.

Project Details

Category: Web Development

Client: BCIT School Project

Date: 2016 December

Project Type: Individual Project

Purpose:

  • Use React to create real-time newsfeed pages like facebook newsfeed.
  • Use Facebook API to allow people to leave messages with their facebook account.
  • Store the messages in the database for other users to see.

Outcome:

  • Use React js to make the content update automatically without having to refresh the page.
  • Implement Google API for the user to pick their location rather than typing out the location manually in the input.
  • Use Facebook API to grab user's information which auto-fills the inputs with the facebook information.
  • Using AJAX to store content in the database within the React JS.
