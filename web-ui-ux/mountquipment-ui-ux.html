MOUNTQUIPMENT UX

User Interface and Experience Documentation

Description

This is the UI and UX documentation for the Wordpress Project called Mountquipment. While creating a simple website on Wordpress I was required to create UI and UX documents to record the process.

Process

First of all, I created two personas for this project. From the personas, I then created task models to create a sample user behavior pattern. When these two were finished, I created sitemaps and different versions of wireframes to create a user-friendly website. Following up from the wireframes, I then created a style tile mock up for the website. After the website was created, a usability test was done and was documented properly.

Summary

From this project, I was able to learn the full process of different UX and UI stages. Overall, I had a better understanding of web development process by the end of this project.

Project Details

Category: Web UI & UX

Client: BCIT School Project

Date: 2016 May

Project Type: Individual Project

Purpose:

  • Create effective personas.
  • With the personas create task models.
  • Make mock ups and wireframes.
  • Report usability testing report.

Outcome:

  • Understand a full process documentation that is required when creating a website.
  • Use Adobe Reader to edit documents.
