PORTRAIT DRAWING

Adobe Photoshop

Description

The goal of this project was to paint a real life photo in Photoshop. Some of the advanced techniques of using brushes were used for shading and creating patterns for facial features.

Process

I started this project by drawing the outline of the photograph using the pen tool. Afterward, using the brush tool, I separated the individual portion of the photographs for highlights and shadows. The specific facial features took a long time to process and create a detailed look. Most of the time were spent on the eyes, mouth, and the hair. When all features were finished, I began shading the highlights and shadows for a smooth transition.

Summary

With this project, I became proficient with using brushes and creating my custom brushes in photoshop. This project was difficult due to the time it took to capture all the details that the photo provides. However, it was a unique experience to paint a picture with just photoshop.

Project Details

Category: Graphic Design

Client: BCIT School Project

Date: 2016 December

Project Type: Individual Project

Purpose:

  • Create a greyscale painting from scratch with a reference picture.

Outcome:

  • Use brush tool effectively to create highlight and shadows.
  • Incorporate different type of brush using preset brush menu.
